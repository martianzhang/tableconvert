Usage: tableconvert [OPTIONS]

Convert between different table formats (MySQL, Markdown, CSV, JSON, Excel, etc.)

Options:
  --from|-f={FORMAT}     Source format (e.g. mysql, csv, json, xlsx)
  --to|-t={FORMAT}       Target format (e.g. mysql, csv, json, xlsx)
  --file={PATH}          Input file path (or use stdin if not specified)
  --result|-r={PATH}     Output file path (or use stdout if not specified)
  --template={PATH}      Template file path
  --verbose|-v           Enable verbose output
  --mcp                  Run as MCP (Model Context Protocol) server
  -h|--help              Show this help message
  --help-formats         Show all supported formats and their parameters
  --help-format={FORMAT} Show parameters for a specific format (e.g. --help-format=markdown)

Data Transformation Options:
  --transpose            Transpose the table (swap rows and columns)
  --delete-empty         Remove empty rows from the table
  --deduplicate          Remove duplicate rows
  --uppercase            Convert all text to UPPERCASE
  --lowercase            Convert all text to lowercase
  --capitalize           Capitalize the first letter of each cell

Format-Specific Options:
  Each format supports its own extension parameters. Examples:
    --align=l,c,r         For markdown: column alignment
    --style=box           For ascii: table style (box, plus, dot, bubble)
    --delimiter=TAB       For csv: value delimiter
    --format=object       For json: output format (object, 2d, column, keyed)
    --minify=true         For html/json/xml: minify output
    --table=mytable       For sql: table name
    --template=file.tmpl  For tmpl: template file path

Examples:
  # Convert from CSV to JSON
  tableconvert --from=csv --to=json --file=input.csv --result=output.json
  # Read from stdin and write to stdout
  cat input.csv | tableconvert --from=csv --to=json
  # Convert from MySQL to Markdown with options
  tableconvert --from=mysql --to=markdown --file=input.csv --bold-header --align=l,c,r
  # Transpose and capitalize
  tableconvert --from=csv --to=markdown --file=input.csv --transpose --capitalize
  # Show help for markdown format
  tableconvert --help-format=markdown
  # Show all formats and their parameters
  tableconvert --help-formats
  # Run as MCP server for AI assistants
  tableconvert --mcp
